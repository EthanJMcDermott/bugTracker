Company Code

The company code is a 6 digit code generated when a new company is registered on Bug Tracker. Simply input this code when registering your account to get started with your company.
To find the company code go to your user profile where you can also change company codes.

Register a New Company

Register a new company to create a blank canvas for your all of your bug tracking needs. The user registering the company will be assigned the role of admin.

Demo Project

By default, all accounts are registered to the demo company. If no company code is given and a new company is not registered, the demo company page will be loaded upon login. Change to or from the demo page in the user profile page.
